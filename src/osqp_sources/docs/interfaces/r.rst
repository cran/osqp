.. _R_interface:


R
========

R interface is still experimental. Here is an example of how it works:

Example
-------

.. code:: r

  library(rosqp)
  ## example, adapted from ?quadprog::solve.QP
  Dmat       <- diag(3)
  dvec       <- c(0,-5,0)
  Amat       <- matrix(c(-4, 2, 0, -3, 1, -2, 0, 0, 1),3,3)
  bvec       <- c(-8,2,0)
  res = solve_osqp(Dmat, dvec, Amat, bvec)
  print(res$x)
